# TAGIRON assistance

Do not use this if you just want to have fun.

## Requirements

- macOS or Linux
- Python 3.10 or higher
- Poetry

```sh
$ poetry install
# may take a while...

$ poetry run python main.py
```

## Terms

- Tiles (数字タイル)
- Question cards (質問カード)

## Input formats

- `[1-9][0-9]*` number
- `<number>(\s<number>)*` list of numbers
  - `none` empty list
- `exit` exit code, bring back to main menu
